Basic Form

        URL component:
        http://www.abc.com:80/contact/?key=value&key=value

        -http: protocol
        -www: domain prefix
        -abc.com: domain
        -80: port number of http
        -if HTTPS: S-SSL: secure socket layer:port: 443
        -Therefore:HTTPS: uses two parts: 80 for localhost
        and 443 for S-SSL
        -contact: slug/pathname
        - ?key=value: Query String parameter
        -URL: Unified Resource Identifier
            Type:-URL: Locator
                 -URN: Name
